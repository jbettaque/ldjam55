# Game Code for LudumDare 55

## Installation

1. First, you need to install the game engine [love2d](https://www.love2d.org/).

2. Afterwards, the game can be run using either love or one of the provided scripts:

   ```shell
   # on windows
   .\run_game.bat

   # on linux
   ./run_game.sh

   # with love directly
   love ./
   ```
